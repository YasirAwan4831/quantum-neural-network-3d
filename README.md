## 🚀 Overview

**Quantum Neural Network** is a high-performance, interactive **3D visualization system** that simulates neural structures using advanced WebGL rendering and custom GLSL shaders. The project combines **mathematics, graphics programming, and UI/UX design** to create a futuristic experience where users can explore dynamic network topologies in real-time.

It is not just a visual — it's an **interactive simulation engine** with energy pulses, morphing structures, and immersive animations.

---

## ✨ Core Features

### 🧠 Neural System

* Real-time **3D Neural Network Rendering**
* Intelligent **node & connection system**
* Smooth animated **data flow simulation**
* Dynamic pulse-based interaction system

### 🔄 Formation Modes

* 🌐 **Crystalline Sphere** (Golden Ratio Based)
* 🧬 **Helix Lattice (DNA Style)**
* 🌿 **Fractal Web (Recursive AI Pattern)**

### ⚡ Interactivity

* Click → Send **energy pulses**
* Drag → Explore 3D space
* Morph → Change network structure
* Freeze → Pause simulation

### 🎨 Visual System

* Bloom Glow Effects ✨
* Particle Starfield 🌌
* Glassmorphism UI Panels
* Smooth camera transitions

---

## ⚙️ How It Works

### 1. Initialization

* Scene, Camera & Renderer setup
* Post-processing (Bloom Effect)
* Network generation

### 2. Rendering Pipeline

* Geometry → Shader → GPU
* Nodes & Connections rendered via BufferGeometry
* Effects applied using EffectComposer

### 3. Interaction Engine

* Raycasting detects user clicks
* Pulse waves propagate across nodes
* Real-time shader updates
